Detailed Description

Represents the main window of the application.

This class inherits from QMainWindow and provides a user interface for interacting with a sensor array connected to a serial port. The class includes several member functions and variables for handling user input, drawing graphics on the window, and communicating with the serial port to receive sensor readings.

◆ readSerialData

void MainWindow::readSerialData ( )
privateslot

Handles the data received from the serial port.

Reads data from the serial port and updates the progress bars and value labels.

If the reading data flag is set to false, this function returns immediately without doing anything. This function attempts to read one line of data from the serial port and splits it into sensor values. If the number of sensor values is 5, it updates the corresponding progress bars and value labels. It also highlights the label with the highest value in green.
