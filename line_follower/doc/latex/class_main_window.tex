\doxysubsection{Detailed Description}
Represents the main window of the application. 

This class inherits from QMain\+Window and provides a user interface for interacting with a sensor array connected to a serial port. The class includes several member functions and variables for handling user input, drawing graphics on the window, and communicating with the serial port to receive sensor readings.
